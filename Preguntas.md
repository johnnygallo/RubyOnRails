## 🔐 Pregunta 1: Seguridad y Autenticación

Una aplicación web puede utilizar autenticación tradicional basada en sesiones o autenticación mediante **JSON Web Tokens (JWT)**.

### 📝 Analice las siguientes preguntas:

* 🔑 ¿Cuáles son las principales ventajas de utilizar JWT en comparación con las sesiones tradicionales?
* 🌐 ¿En qué tipo de aplicaciones considera más apropiado utilizar JWT?
* ⚠️ ¿Qué riesgos de seguridad podrían existir si un token JWT es robado por un atacante?
